Lag kost / benchmark plot

Scenario analyse med kjpte modeller og bruk av API-tilgang

Analysen antar:

  • 800 ansatte
  • De sender mellom 50 og 350 requests per dag (inkl. automatiserte prosesser som gjr requests) til modellenes API
  • 50 ord og 350 ord per sprsml
  • 100 til 600 ord per svar
  • 1.33 tokens per ord

Spesielt requests antas vre konservativt da et automatisert program som gjr klassifisering eller datainnhenting kan medfre mange fler requests. Sprsmlslengden er ogs svrt konservativ da konteksten fra selv enkle RAG modeller gjr at man potensielt fr mye lengre sprsml enn 350 ord (spesielt hvis man prver oppsummere dokumentasjon). Tokens per ord kan ogs variere med hvilken embedder-modell som brukes og hvilket sprk man sender sprsml p.
